# BMI-calculator
BMI Calculator Android Application built using Flutter and Dart. This project calculates Body Mass Index (BMI) using both American and Indian standards and provides health recommendations based on the result. Includes project report, system architecture, use-case diagram, flow diagram, and core Dart source files.

A simple and interactive BMI (Body Mass Index) Calculator Application developed using Flutter and Dart as part of a Computer Science Engineering mini-project.

This application calculates BMI using standard formulas and categorizes the result based on WHO ranges, along with American and Indian standards. It provides users with clear health recommendations, making it easy to understand their fitness level.
